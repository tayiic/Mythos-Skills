Anthropic skill packages are clean-room minimal optimized versions. `ORIGINAL.md` records public source metadata and benchmark baselines, but does not copy proprietary source text.

Superpowers skill packages preserve the workflow idea in minimized form and link to the MIT source. Full source text is not copied into this first batch.
