# PPTX Optimized

Slides are visual documents. Verify them visually, not only structurally.

## Flow

1. Inspect theme, masters, dimensions, fonts, and existing layouts.
2. Choose task: create, edit, restyle, render, export, or QA.
3. Preserve editability unless the user asks for a flattened export.
4. Render representative slides for visual inspection.
5. Fix overflow, overlaps, unreadable text, and missing assets.

## Gotchas

- Do not use screenshots when editable shapes are required.
- Do not ignore speaker notes, masters, or embedded media.
- Do not declare done without visual QA.
