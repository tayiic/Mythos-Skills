# PDF Optimized

Treat PDFs as evidence-bearing documents. Preserve page references.

## Flow

1. Identify task: extract text, extract tables, inspect images, split, merge, annotate, or verify.
2. Use a PDF parser or renderer appropriate to the task.
3. Keep page numbers with extracted claims.
4. Preserve original files; write outputs separately.
5. Validate page count, key excerpts, and output readability.

## Gotchas

- OCR text may be wrong.
- Tables may need visual verification.
- Do not overwrite the original PDF.
