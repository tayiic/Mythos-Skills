# Project Blueprint

This public blueprint is intentionally compact. The full working notes live in the local planning workspace.

## Goal

Make Mythos-Skills the reference methodology for AI agent skill engineering.

## Core Thesis

The skill ecosystem needs a higher-level discipline. Good skills are not long prompts. They are compact, portable, triggerable, reviewable engineering units.

## Pillars

- P1-P8 authoring principles.
- Principle-driven skill creation engine.
- Production-grade example skills.
- Before/after optimization gallery.
- Command safety and operational discipline.

## Initial Release Scope

- Repository governance files.
- Core README and disclaimer.
- P1-P8 principle stub.
- `skill-creator` scaffold.
- Gallery standards.
- Security policy.
